# Buildo

This is my little Markdown YAML parser, thing.

## Commands

```
php buildo hobbiton:add                 # Coming Soon - adds a new site (interactive)
php buildo hobbiton:remove <site>       # Coming Soon - removes site
php buildo hobbiton:edit <site>         # Coming Soon - edits site

php buildo hobbiton:clean <site>        # Coming Soon - clears the "compiled" list

php buildo compile <site>               # Compiles all the pages
php buildo compile <site> --nocache     # Coming Soon - this won't save to the compiled list
php buildo compile <site> --force       # Coming Soon - this will ignore the cache
```
